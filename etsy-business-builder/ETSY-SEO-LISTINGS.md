## SEO TIPS FOR ALL LISTINGS

1. Use all 13 tags on every listing (Etsy allows exactly 13)
2. Front-load the title with highest-search-volume keywords
3. First 3 lines of description appear in search results - make them count
4. Add "Instant Download" and "Printable" in titles where space allows
5. Use variation keywords in description body (synonyms, related terms)
6. Update tags seasonally (e.g., add "new year" tags in December)
7. Add mockup images showing the PDF on tablet/printed
8. Include a "What's Inside" section with page count in description
9. Mention "No physical product shipped" in description
10. Add customer testimonials to description after first sales
