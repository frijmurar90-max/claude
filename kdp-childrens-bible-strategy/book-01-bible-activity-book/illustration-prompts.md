## GLOBAL STYLE GUIDELINES

**For All Illustrations:**
- Style: Bold cartoon line-art, black outlines on white background
- Line weight: Thick outlines (2-3pt minimum), suitable for children's coloring
- Shapes: Simple, rounded, child-friendly forms with large enclosed coloring areas
- Age target: 4-8 years old — no tiny details or intricate patterns
- Resolution: 300 DPI minimum, vector preferred
- File format: PNG or SVG with transparent background
- Page bleed: None (no elements touching page edges)
- Safe margin: Keep all elements within 0.5" from trim edges

**AI Image Generation Universal Prefix:**
> "Children's coloring book page, black and white line art, thick bold outlines, simple shapes, large coloring areas, no shading, no gray tones, no color fills, kid-friendly cartoon style, ages 4-8, 8.5x11 portrait orientation"

## PRODUCTION NOTES FOR ILLUSTRATOR/AI GENERATION

### File Specifications:
- **Interior pages:** 300 DPI, 8.5" × 11", black and white only (no grayscale)
- **Cover:** 300 DPI, CMYK color, dimensions per KDP cover calculator (include spine width based on page count)
- **File format:** PDF (print-ready) or high-res PNG
- **Bleed:** None for interior; 0.125" for cover

### Style Consistency Rules:
1. All human figures should have the same proportions (large heads, round faces, dot eyes or simple eyes)
2. Line weight must be consistent throughout (2-3pt for outlines, 1pt for interior details)
3. All animals should be friendly/cartoonish (never realistic or scary)
4. Biblical figures should be modest, simple robes, no graphic violence
5. Diversity: Include various skin tones and hair types in group scenes
6. Avoid any denominational-specific symbols (keep broadly Christian)

### AI Generation Tips:
- Add "coloring book page" to every prompt for interior pages
- Add "no gray, no shading, no fills, pure black lines on white" to avoid AI adding tones
- If AI generates too much detail, add "simplified, kindergarten level, very simple shapes"
- For consistent characters, describe them the same way each time (e.g., "Jesus: kind-faced man with shoulder-length hair, simple robe, gentle smile")
- Generate at highest resolution available, then clean up in vector software if needed
